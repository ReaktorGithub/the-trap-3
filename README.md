# Моя первая игра на Unreal Engine

Версия движка: 5.7.4

Жанр: платформер от 3 лица

Задача: Соберите все кольца на уровне, чтобы открыть дорогу к выходу! Остерегайтесь хитроумных ловушек и врагов, решайте несложные загадки.

Время на прохождение: 10-20 мин

Управление: клавиатура, мышь, геймпад

Системные требования: затрудняюсь сказать, но должно работать на среднем пк
